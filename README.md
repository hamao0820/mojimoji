# もじもじ

上から降ってくる「もじ」を揃えて単語を作って行くゲームです。

## 準備

-   Clone the repository

```zsh
git clone https://github.com/shunsukehamada/mojimoji.git && cd mojimoji
```

-   インストール

```zsh
npm install
```

## 実行

```zsh
npm run dev
```
